What are Number Systems?
A number system is, by definition, a system of writing to express numbers; it consistently represents numbers of a given set using digits or other symbols. This provides a distinct representation of every number and represents the arithmetic and algebraic structure of figures. It also allows us to perform arithmetic operations like addition, subtraction, multiplication, and division (Admin, 2023).

The value of any digit in a number can be determined by:
- The digit
- Its position in the number
- The base of the number system

Binary Number System
A binary number system has a base of 2 since it only uses two digits from 0 to 1, so numbers are expressed in a sequence of ones and zeros. Each binary digit or bit corresponds to the absence or presence of an electrical charge in a computer’s memory or storage device. Binary is used in a computing system as the foundation for binary code, which uses unique combinations of ones and zeros to represent numbers, letters, and other digital information (Thomas, 2025). Applications of Binary Number System:
-Computer Processors
-File Compression and Decompression
-Cryptography
-Digital Video and Audio Processing
-Machine Learning and AI
Decimal Number System
A decimal number system has a base of 10 since it uses ten digits from 0 to 9. The positions successive to the left of the decimal point represent ones, tens, hundreds, thousands, and so on. We use this number system more in our day-to-day lives, when dealing with money, weight, length and etc. These are situations where more precision is required than whole numbers can provide (GeeksforGeeks, 2025). Applications of Decimal Number System:
-Banking and Finance
-Cooking and Recipes
-Length and Weight
-Sports and Fitness
-Health and Medicine
-Travel and Navigation
-Science and Engineering
Octal Number System
The octal number system has a base of 8, for it uses digits from 0 to 7 to represent numbers. This number system aids programmers in efficiently translating between binary and decimal. Making it easier to write and understand low-level code for computers, laptops, and desktops (Octal Explained: The Programmer’s Base-8 Code, 2023). Applications of Octal Number System:
-File Permissions
-Memory Addressing
-American Standard Code for Information Interchange (ASCII)
Hexadecimal Number System
The hexadecimal number system, which has a base of 16 using digits 0 to 9 and letters A to F, each letter having a numerical value of 10 to 15. Programmers also use this number system to help them simplify and write large binary numbers in just a few digits (Iannuttall, 2024).

Applications of Hexadecimal Number System:
-Defining locations in memory
-Defining colors on web pages
-Representing Media Access Control (MAC) addresses
-Displaying error messages
